# スライド01 表紙
皆川様、本日はご覧いただきありがとうございます。株式会社シスコムネットの寺嵜より、先日の省エネルギー診断の結果を踏まえ、KIZUNA HOME様の電気代削減のご提案を、要点を絞ってお伝えいたします。

# スライド02 現状の電気代と課題
まず現状です。KIZUNA HOME様の年間電気代は約700万円、年間使用量は約18万キロワットアワー。24時間稼働で、特に冬の早朝に暖房と給湯のピークがあります。入居率はまだ約30パーセントですが、満床に近づくと電気代は約2,100万円規模まで増える見込みです。だからこそ、減らす、作る、貯める、の3段階が効いてきます。

# スライド03 3段階で電気代を下げる
ご提案の柱は3段階です。第1に、窓の遮熱・断熱コーティングで熱の出入りを抑え、電気を減らす。第2に、太陽光発電78.75キロワットで電気を作る。第3に、蓄電池40.5キロワットアワーで貯めて、夕方から夜に使う。いずれも6/3の省エネ診断（レナリア）の結果を踏まえた組み合わせです。

# スライド04 2つのプラン（A・B）
ご提案は2つのプランです。Aプランは補助金最適化を図った構成で、投資総額2,167万円、補助金496万円を差し引いた自己負担は1,671万円、投資回収は現状ベースで6.0年。Bプランは削減最大化を狙い、自己負担2,448万円、回収7.6年です。どちらを選ばれても正解になるよう設計しております。

# スライド05 電気代削減効果と投資回収
電気代の削減効果です。現状の入居率では年間でAプラン277万円、Bプラン322万円。満床時にはAプラン408万円、Bプラン558万円まで広がります。投資回収は満床達成で4.1年から4.4年程度に短縮し、20年間の累計では最大で約8,712万円の効果が見込めます。入居率の改善と一緒に進めるほど、効果が高まります。

# スライド06 停電時の備え（BCP）
金額に表れない価値が、停電時の備えです。認知症対応型施設のご入居者様は環境の変化が大きな負担となり、停電や避難は健康リスクに直結します。本提案の蓄電池40.5キロワットアワーは、停電時に特定の回路を12から24時間バックアップし、太陽光と連携すれば晴天時は無期限で自立できます。ご入居者様とご家族の安心に直結いたします。

# スライド07 進め方と補助金スケジュール
進め方です。スケジュールは、2026/5/15に受付開始（済）、6/3に省エネ診断 実施（済）、〜7月に申請書類の整備、10/7に申請締切、11〜12月に交付決定後に工事、という流れです。ひとつだけご注意は、交付決定の前に契約・着工すると補助対象外。申請の手続きは寺嵜が代行いたしますので、ご負担は最小限です。
