# 寺嵜流 提案メソッド（減らす・作る・貯める）

提案の数値ロジックと論法の根拠。KIZUNA HOME案件から確立。更新したら日付を添える。

## 計算式（diagnose.py と一致させること）

- **自己負担** = 投資総額 − 補助金見込
- **投資回収（現状）** = 自己負担 ÷ 年間削減額（現状入居率）
- **投資回収（満床）** = 自己負担 ÷ 年間削減額（満床/フル稼働）
- **20年累計純効果** = 年間削減額（満床） × 20 − 自己負担
- 評価年数は20年で固定（機器の想定稼働年数に合わせた寺嵜流の標準）。

> 年間削減額は省エネ診断の実測・試算から入れる。回収年数・累計は機械的に導出する（盛らない）。

## 3段階の柱

1. **減らす** — 窓の遮熱・断熱コーティング（例: ビッグファインG）。熱の出入りを抑える。
2. **作る** — 太陽光発電（kW）。年間発電量(kWh)を提示。
3. **貯める・使う** — 蓄電池（kWh）。夕方〜夜間のピークに充当＋停電時BCP。

## 論法の順番（崩さない）

現状の可視化（満床時いくらに増えるかまで） → 3段階 → A/B二択（どちらも正解） →
削減効果の幅（現状/満床） → 金額に出ない価値（停電時の安心を顧客特性に紐づけ） →
補助金スケジュール（交付決定前の契約・着工は対象外、申請は寺嵜が代行） →
まとめ3点（下がる・抑える・守る）＋連絡先。

## A/Bプランの設計思想

- **Aプラン = 補助金最適化**: 補助対象を最大限活かし自己負担と回収を軽く。
- **Bプラン = 削減最大化**: 対策範囲を広げ削減額・累計効果を最大化。
- 「どちらを選ばれても正解になるよう設計」と必ず添える（決定の心理的負荷を下げる）。

## 口調

丁寧・誠実・押しつけない。「お客様のペースで」「最後まで伴走」。数字は必ず根拠とセット。
