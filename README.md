## 使い方（最短）

`index.html` をダブルクリックしてブラウザで開く → 「▶ ナレーション付きで開始」を押すだけです。
**MP3の準備は不要**です（ブラウザ内蔵音声で再生されます）。

| 操作 | 内容 |
|---|---|
| ▶ 再生 / 停止 | 下部中央の再生ボタン（または **P** キー） |
| 次のスライド | → / スペース / 右スワイプ |
| 前のスライド | ← / 左スワイプ |
| 目次 | 右下のメニューボタン |
| 字幕 ON/OFF | 左下の字幕ボタン |
| 音声モード切替 | 左下の🔊ボタン（ブラウザ音声 ⇔ MP3） |

ナレーション再生が終わると自動で次のスライドへ進みます。

---

## 2つの音声モード

| モード | 内容 | おすすめ |
|---|---|---|
| **ブラウザ音声** | Chrome / Edge / Safari 内蔵の日本語音声。高品質・ネット接続でフォントも綺麗。 | ◎ 皆川様向け推奨 |
| **MP3ファイル** | 同梱MP3を再生。完全オフライン対応だが音質はやや機械的。 | ○ ネット不可環境の予備 |

MP3モードを使う場合のみ、`ナレーションMP3生成.ps1` を実行して `audio/01.mp3〜25.mp3` を用意してください（手順は `実行手順_PowerShell.md`）。
MP3が無い状態でMP3モードにした場合は、自動でブラウザ音声に切り替わります。

## GitHub Pagesでの公開手順

皆川様にURLでお渡しする場合の手順です。

1. GitHubで新規リポジトリを作成（Public、例：`kizuna-proposal`）
2. このフォルダの中身（`index.html`、`audio/` など）をアップロード
3. Settings → Pages → Branch: `main` / `root` → Save
4. 数分後に発行されるURL（例：`https://t-terasaki-beep.github.io/kizuna-proposal/`）を皆川様へ
